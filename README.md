# Student Performance Analysis using SQL

This project analyzes a dataset of computer science students using SQL queries.

## Dataset
The dataset contains information about students including:
- Name
- Gender
- Age
- Major
- GPA
- Number of Projects
- Programming Skills (Python, SQL, Java)

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- Aggregate Functions (COUNT, AVG, MAX, MIN)
- ORDER BY
- LIMIT
- Subqueries
- Window Functions

## Window Functions Used
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- PARTITION BY
- LAG()
- LEAD()

## Analysis Performed
- Top students based on GPA
- Ranking students by performance
- Students with highest number of projects
- Students above average GPA
- Major-wise performance analysis
